# LP Wallet Tracker 📊

Еженедельный мониторинг LP позиций детей с отчётами в Telegram.

## Функционал

- 📅 Еженедельные отчёты (воскресенье 10:00 MSK)
- 💰 План vs Факт для каждого ребёнка отдельно
- 📈 Изменения за неделю (абсолютные и %)
- 📆 Изменения за месяц (первая неделя каждого месяца)
- ✅ Проверка активов на соответствие белому списку
- 👛 Два кошелька: Марта + Аркаша

## Кошельки и планы

| Имя | Адрес | План март 2026 |
|-----|-------|----------------|
| Аркаша | `0x305220d077474c5cab839E7C1cB3264Aca19f1B9` | $12,685 |

## Настройка

### Secrets (GitHub → Settings → Secrets)

```
TELEGRAM_TOKEN     - Токен бота
TELEGRAM_CHAT_ID   - ID чата для отправки
DEBANK_API_KEY     - API ключ DeBank (опционально)
```
